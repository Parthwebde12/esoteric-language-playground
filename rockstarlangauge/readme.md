# Rockstar Playground

Exploring Rockstar, an esoteric programming language where code looks like song lyrics.

## Hello World

```rock
My heart says "Hello, World!"
```

## Features

- English-like syntax
- Programs resemble song lyrics
- Fun and easy to read
- Great introduction to esoteric programming languages

## File Extension

`.rock`

Part of my "10 Days of Esoteric Programming Languages" challenge.
